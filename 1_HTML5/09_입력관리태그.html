input태그

웹 문서에서 사용자가 입력할 수 있는 양식을 제공하는 태그

text 관련된 input 태그

type="text" - 기본값

            - 한 줄 짜리 문자열(텍스트)를 입력할 수 있는 입력 상자
            - input 태그의 type 속성의 기본 값으로, 생략가능
            - inline형식(모든 input 태그가 inline)
        
            
text 관련 input 태그의 공용 속성

            - size : 입력상자의 크기
            - maxlength : 입력받는 텍스트의 최대길이
            - placeholder : 입력상자에 작성될 내용을 설명

form태그

            - div와 같은 영역 태그(block 형식)
            - 내부에 작성된 input 태그의 값을 서버로 전달하는 역할
            
            [속성]
            1) action : 내부에 작성된 값을 전달할 서버 또는 페이지 주소를 작성
                (미작성 시 기본값으로 현재 페이지의 URL 적용)
            2) name : form 태그의 이름을 지정하여 form 태그 구분(JS 사용)
            3) method(방식) : 서버로 데이터 전달하는 방식을 지정하는 속성
                - GET/POST 지정 가능(기본값 GET)
                - get 방식 : 서버로 입력 값을 보낼시 그대로 url에 표시
            4)target : action의 지정도니 주소를 현재탭/새탭 중 어디서 열지 지정
