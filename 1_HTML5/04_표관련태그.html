표 관련 태그

table 태그

- 웹 문서에서 자료 정리를 위해 주로 사용하는 태그
- 행과 열로 이루어져 있으며, 행과 열이 교차하는 지점을 셀(cell)이라고 함.
- table 태그는 테이블을 나타내는 행과 열이 작성될 수 있는 영역을 지정함.
        
tr 태그

- Table Row의 약자로 한 행을 타나내는 태그

        
td 태그

- Tabel Data의 약자로 한 행의 한 컬럼(== 셀)을 나타내는 태그
        
th 태그

- Table Header 약자로 컬럼명을 표시하는 용도의 태그
- 기본 성질은 td태그와 같으나 추가적으로 굵은 글씨, 가운데 정열이 되어 출력이 됨

테이블 구조 설정 태그

thead : 테이블의 상단 부분 영역(컬럼명)
tbody : 테이블의 중단 부분 영역(실제 값, 내용)
tbbot : 테이블의 한단 부분 영역(합계)

해당 영역을 나누지 않을시 스타일을 일관적으로 주고싶을 때 힘들기에 사용
